## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ cd nestjs-typeorm-postgresql-swagger
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
Navigate to http://localhost:3000/api and for Swagger http://localhost:3000/api/docs


## Test

```bash
$ cd nestjs-typeorm-postgresql-swagger
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
